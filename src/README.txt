=============================
   VitaConverter - Media Converter
=============================

System Requirements:
- Java JDK 21 or newer
- Windows 7/8/10/11
- 100MB free disk space

Manual Installation:
- Java JDK: https://adoptium.net/
- FFmpeg: https://ffmpeg.org/

Troubleshooting:
- Jika program tidak jalan: Pastikan Java terinstal
- Jika konversi gagal: Pastikan FFmpeg terinstal
- Restart komputer setelah instalasi
